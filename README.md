# A-Hybrid-Deep-Learning-Ensemble-For-Credit-Card-Fraud
A hybrid deep learning ensemble for credit card fraud detection using CNN, LSTM, and Transformers as base learners, with XGBoost as the meta-learner. Achieves 0.961 sensitivity, 0.999 specificity, and 0.972 AUC-ROC on the European Credit Card Dataset, outperforming traditional methods.
